# KB-002: Quote Export to PDF Not Working

## Symptoms
- Clicking "Export to PDF" does nothing
- No file is downloaded
- No visible error message

## Likely Causes
- Browser pop-up blocker is enabled
- Corrupted browser cache
- Temporary export service interruption
- Unsupported browser configuration

## Resolution Steps
1. Ensure pop-up blocker is disabled for the application
2. Clear browser cache and cookies
3. Retry export in a different browser (Chrome or Edge recommended)
4. Check if browser extensions are interfering (disable temporarily)
5. Refresh the page and try again

## Escalate If
- Issue persists across multiple browsers
- Multiple users report the same issue
- Export fails consistently after all troubleshooting steps
