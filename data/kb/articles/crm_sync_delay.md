# KB-003: CRM Sync Delay or Missing Updates

## Symptoms
- New quotes do not appear in CRM (e.g., Salesforce)
- Updates take longer than expected to sync
- Data appears incomplete or outdated

## Likely Causes
- Sync delay due to system load
- API rate limits or throttling
- Integration configuration issue
- Temporary service disruption

## Resolution Steps
1. Confirm that CRM integration is active and configured correctly
2. Wait 10–15 minutes and check again for updates
3. Verify that API credentials are valid and not expired
4. Trigger a manual sync if available
5. Check integration logs if accessible

## Escalate If
- No sync occurs after 15–30 minutes
- Duplicate or inconsistent records appear
- Multiple users or accounts are affected
