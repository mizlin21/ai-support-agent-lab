# KB-005: Browser Cache or UI Rendering Issues

## Symptoms
- UI elements appear broken or misaligned
- Buttons overlap or do not function
- Dashboard does not load correctly

## Likely Causes
- Cached browser data causing conflicts
- Unsupported browser version
- Browser extensions interfering with UI

## Resolution Steps
1. Clear browser cache and cookies
2. Refresh the page after clearing cache
3. Try accessing the platform in an incognito window
4. Test in a different browser (Chrome or Edge recommended)
5. Disable browser extensions and retry

## Escalate If
- Issue persists across multiple browsers
- UI problems affect multiple users
- Rendering issues occur after a recent release
